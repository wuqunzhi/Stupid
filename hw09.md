## 3
### 类和对象的区别：
类是抽象的，对象是具体的，类包括对象，对象是类的实例。  
如：牧师（对象）包含于sprite（类）中

## 4
### 魔鬼的属性和方法：
属性：图片名字，大小，动作，初始位置。  
方法：  
1、点击时跳入船中或从船上回到岸上。  
2、牧师少时执行杀死牧师。  
3、点击go按钮划船。

## 5、假设魔鬼被鼠标点中，会执行onclick事件，请用文字 （伪代码）描述这个事件中魔鬼与其他对象沟通的过程。
```
BEGIN
if 点击魔鬼 then
    if 魔鬼在岸上 then
        魔鬼跳下船
    end if
    else 魔鬼上岸
        if 岸上魔鬼多于牧师 then
            魔鬼执行杀死牧师动作
            游戏结束弹出try again按钮
        end if
    end else    
end if
END
```
